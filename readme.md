# 趣味科普小百科

这是一个面向小朋友和家长的科普小百科，旨在通过生动有趣的方式介绍科学知识。

## 创作原则

- 面向小朋友和家长
- 有趣的科普小知识
- 培养孩子善良、诚实等好品质
- 展现生活的美好，发现世界的奇妙
- 生动有趣，简单易懂
- 配有精美插图
- 不说教，不枯燥
- 每个主题独立成目录
- 图片大小适中，适合移动端阅读
